# Contributing to SFT-ISU Student Portfolios

Thanks for wanting to add your portfolio! Follow these steps and your PR will get merged quickly. 🎉

---

## ✅ Before You Submit

Make sure **all** of the following are true:

- [ ] You are a **current student or alumnus of ITM SKILLS UNIVERSITY**
- [ ] Your **GitHub profile bio** mentions `SFT-ISU` — this is how we verify you belong here
- [ ] Your portfolio website is **live and accessible**
- [ ] You haven't already submitted

---

## 📋 Submission Format

Add your entry inside the correct **Batch → Cohort** section in `README.md`, in **alphabetical order by first name**.

```
* [Your Full Name](https://your-portfolio-url.com)
```

## 🔧 Step-by-Step: How to Open a PR

1. **Fork** this repository
2. **Clone** your fork:
   ```bash
   git clone https://github.com/YOUR_USERNAME/sft-isu-portfolios.git
   ```
3. **Create a new branch**:
   ```bash
   git checkout -b add-your-name
   ```
4. **Edit `README.md`** — find your Batch → Cohort section and add your entry in alphabetical order
5. **Commit**:
   ```bash
   git add README.md
   git commit -m "Add [Your Name] — Batch XX-XX, [Cohort Name] Cohort"
   ```
6. **Push**:
   ```bash
   git push origin add-your-name
   ```
7. **Open a Pull Request** — the PR template will guide you

---

## ❌ PRs Will Be Rejected If

- Your GitHub bio does **not** mention `SFT-ISU`
- The portfolio link is broken or leads to a blank page
- The entry is not placed in the correct Batch → Cohort section
- The entry is not in alphabetical order within the cohort
- The format doesn't follow the template
